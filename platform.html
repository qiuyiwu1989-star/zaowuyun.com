02_// capabilities in detail

每一层展开看

L1 / models

模型层

既要"用最强的"也要"用得起" —— 路由 + 评测 + 自研三件套。

/01

多模型路由

Claude / GPT / 文心 / 通义 / Qwen / DeepSeek 智能路由 + 成本治理。

/02

自研行业大模型

工业设计、商品摄影、儿童认知三个细分场景的领域大模型。

/03

私有部署适配

公有云 / 专有云 / 私有化 / 端边混合 —— 4 种部署模式可选。

/04

模型评测平台

业务真实场景的多维度自动评测 —— 不是 benchmark 比赛。

L2 / knowledge

知识层

不是 prompt 库，是企业级的资产 + 行业 Know-How 沉淀。

/01

企业资产库

3D 模型 / 材质 / 贴图 / Logo / 品牌字体 / 历史素材统一治理。

/02

多模态 RAG

文字 / 图片 / 音视频 / 3D 模型统一向量化检索。

/03

行业知识图谱

4 大行业的 SKU 分类、品类知识、工艺关联图谱。

/04

检索质量评估

召回率 / 精确率 / 业务相关度持续监控。

L3 / agents

Agent 编排

让 AI 不止"回答" —— 还能调用工具、长任务、和人协作。

/01

可视化工作流

拖拽式 Agent 流程设计 —— 业务方也能参与。

/02

Tool / MCP 接入

兼容 Anthropic MCP 协议 + 自有工具库 200+。

/03

多 Agent 协作

市场 / 品牌 / 设计 / 营销 4 角色 Agent 组队完成长任务。

/04

监控与回放

每次 Agent 调用都可回放 —— 调试 + 合规审计两不误。

L4 / industry templates

行业模板

每个行业都有自己的"开机即用"模板包。

/01

快消 · 节日模板

双 11 / 春节 / 618 / 七夕 / 儿童节 30+ 节日模板。

/02

工业 · 设计模板

AI 草图 / 多视角 / 工业渲染大片预设。

/03

教育 · 认知模板

开放式探索 / 小实验 / 故事化讲述 / 家长报告。

/04

时尚 · 模特模板

虚拟模特 / 风格趋势 / 季节系列上新自动化。

L5 / data loop

数据闭环

每一次客户使用，都让下一个 AI 应用变强。

/01

标注与评分平台

业务用户的评分回流模型，不依赖外包标注。

/02

在线评测对比

新版本上线前在线 A/B，效果数字说话。

/03

微调与回写

行业大模型按月迭代，客户数据脱敏后回写。

/04

业务指标治理

从模型 metrics 到 business KPI 全链路追踪。

L6 / delivery

应用交付

从设计到运营，"最后一公里"是我们的标配。

/01

多端 UI 组件库

Web / 小程序 / iPad / 平板 / 桌面 端一致体验。

/02

企业版工作台

权限 / 资产库 / 任务中心 / 财务计费一体化。

/03

合规与内容安全

生成式 AI 备案 / 数据隔离 / 审计日志。

/04

运营 SDK

营销活动、推送、用户分层、留存看板。
