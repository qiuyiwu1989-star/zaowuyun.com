# 造物云官网 (zaowuyun.com)

造物云官方网站静态站点源码。

## 目录结构

- `index.html` — 首页
- `about.html` — 关于我们
- `platform.html` — 平台介绍
- `solutions.html` / `solutions-fmcg.html` — 解决方案
- `apps.html` / `apps-qiuqiu.html` — 应用
- `cases.html` — 案例
- `developers.html` — 开发者
- `academy.html` — 学院
- `blog.html` / `news.html` — 博客与资讯
- `contact.html` — 联系我们
- `assets/` — 图片、样式、脚本等静态资源

## 本地预览

直接用浏览器打开 `index.html`，或在项目根目录启动一个本地服务器：

```bash
python3 -m http.server 8000
```

然后访问 <http://localhost:8000>。
